Freevana

Free Python database grabber and updater for Cuevana

Download and maintain a local copy of Cuevana's movies and series complete with download links.

Python Dependencies

mechanize
BeautifulSoup

Authors

 tirino (tirino.wok@gmail.com)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/tirino/freevana
